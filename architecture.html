🔄 Feed Request · End-to-End Data Flow
01 · Browser
10s Poll Fires
Client calls GET /api/feed?page=1 with Bearer JWT every 10 seconds via useEffect interval.
02 · Express
JWT Verified
authMiddleware verifies JWT. req.user populated with user_id. Passes to route handler.
03 · Redis Check
Cache Lookup
Key global:feed:page:1:limit:20 checked in Redis. HIT → return instantly. MISS → go to Postgres.
04 · Postgres
Paginated JOIN
JOIN on posts → users → profiles. Subqueries for like_count & comment_count. LIMIT + OFFSET pagination.
05 · Hydrate Cache
setEx 300s
Result stored in Redis with 5-minute TTL. Future requests within TTL bypass DB entirely — zero cold-path cost.
06 · Personalize
liked_by_me Merge
User-specific liked_by_me flag computed via a single targeted post_likes query and merged into the cached posts.
07 · Response
JSON to UI
Posts with full user metadata, counts, and personalization returned. React re-renders only changed cards.
